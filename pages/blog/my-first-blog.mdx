in this post i’m going to teach you how to create a form with validation by using React.js, Next.js, Typescript, Material-UI, Formik and Yup
First of all we should create a Next.js project so open your terminal on Mac, Windows or Linux

```
$ npx create-next-app perfect-form
then go to the perfect-form project’s folder :
$ cd perfect-form
I call this project perfect-form but you can name it whatever you like,
then we need to install some packages for using the typescript and others
$ yarn add typescript && yarn add --dev @types/node && yarn add --dev @types/react
$ yarn add @material-ui/core
$ yarn add styled-components
$ yarn add --dev @types/styled-components
$ yarn add formik
$ yarn add yup
$ yarn add --dev @types/yup
```

Or if you use npm

```
$ npm install typescript && npm install --dev @types/node && npm install --dev @types/react
$ npm install @material-ui/core
$ npm install --save styled-components
$ npm install --dev @types/styled-components
$ npm install formik
$ npm install yup
$ npm install --dev @types/yup
```

Then in the pages folder change the extension of `_app.js` and `index.js` to tsx and delete the jsx content in return of Home component in index.tsx and type hello form in return of the compnent

```
import { NextPage } from 'next';
import { FC } from 'react';
const Home: NextPage<FC> = () => {
  return (
    <>hello form</>
  )
};
export default Home;
```

To see the result of the project:
`yarn dev`
#or
`npm run dev`

So you will see hello form on `http://localhost:3000`
Next Step: Designing the form
For designing the form we need to use Material-UI and Styled-Component
So I import these components from Material-UI
`import { Box, Button, Card, Grid, TextField, Typography } from '@material-ui/core';`
And for using styled-component
`import styled from 'styled-components';`
Then we use Grid component and Card to create a card in the center of the page

```
const Home: NextPage<FC> = () => {
  return (
    <Grid container justify="center" alignItems="center">
      <Card>
      </Card>
     </Grid>
    )
}
```

Then we add a **Box** in the Card with padding in Card and, I think having a **title for the form should** be nice so, I use [Typography]("https://myways.in") with h4 variant and because we need to put these elements in column, I placed a Grid inside of the Box as a father of Typography and others input element which we will create the in the next part.

```
const Home: NextPage<FC> = () => {
  return (
    <Grid container justify="center" alignItems="center">
      <Card>
        <Box padding={{xs: 5, lg: 10, sm: 7}}>
          <Grid container direction='column'>
            <Typography variant='h4' gutterBottom>
              Join Us
            </Typography>
           </<Grid>
        <Box>
      </Card>
     </Grid>
    )
}
```
